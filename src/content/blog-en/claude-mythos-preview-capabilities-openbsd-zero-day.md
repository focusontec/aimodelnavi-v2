## Introduction

On March 27, 2026, leaked reports revealed that Anthropic is testing its next-generation large language model, Claude Mythos. This model is said to far surpass the scale of its predecessor, Opus, achieving a significant leap in performance. While users expected a lengthy wait for an official release, Anthropic quickly confirmed its existence.

Anthropic announced Claude Mythos Preview and launched Project Glasswing, a cybersecurity cooperation plan. However, this announcement diverges sharply from typical model releases: Anthropic explicitly stated that Mythos will not be publicly available. The reason is its extraordinary capability to explore vulnerabilities and generate exploits—a power so advanced that the company itself harbors serious concerns.

### Development Timeline and Accidental Leak

Notably, internal testing began on February 24, 2026. But in late March, a configuration error in Anthropic's content management system caused a draft blog post to be unintentionally published, thrusting the model into the global spotlight.

### Unmatched Vulnerability Detection: Replicating OpenBSD Zero-Day Exploits

Claude Mythos's true value is highlighted by its advanced security capabilities. A standout demonstration involved discovering and exploiting zero-day vulnerabilities in OpenBSD, an operating system renowned for its robustness. Mythos rapidly identified complex memory errors and deep logical flaws—tasks that were highly challenging for previous models. This transcends basic code analysis, suggesting that AI can autonomously reason through and execute sophisticated hacking techniques.

### Project Glasswing: AI-Powered Defense for the Modern Era

To control these potent capabilities and harness them for societal benefit, Anthropic developed Project Glasswing. This initiative establishes a framework for partnering with select security experts and government agencies to ensure AI-driven vulnerability assessment is used solely for defensive purposes. Anthropic aims to minimize the risk of AI being weaponized in cyberattacks while proactively correcting unknown vulnerabilities through an AI-based defense system.
