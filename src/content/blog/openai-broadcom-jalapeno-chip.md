2026年6月24日、OpenAIとBroadcom（NASDAQ: AVGO）は、OpenAI初の自社設計AI推論アクセラレーター「Jalapeño（ハラペーニョ）」を正式発表した。これはLLM推論に特化して設計された「インテリジェンスプロセッサー」であり、AI業界のインフラスタックに大きな転換点をもたらす。

## Jalapeñoとは

Jalapeñoは、OpenAIがChatGPT、Codex、API、そして今後のエージェント製品で日々稼働しているシステムの知見をベースに、**ゼロから設計されたLLM推論専用チップ**だ。既存の汎用AIアクセラレーターを流用したものではなく、現在および将来のLLMの推論ニーズに合わせて最適化されている。

### 主要仕様

| 項目 | 詳細 |
|------|------|
| チップ名 | Jalapeño（ハラペーニョ） |
| 開発パートナー | OpenAI + Broadcom + Celestica |
| 設計→量産テープアウト | **9ヶ月**（業界最速クラス） |
| 主な用途 | LLM推論（ChatGPT、Codex、API） |
| テスト中のワークロード | GPT-5.3-Codex-Spark |
| 初回デプロイ | 2026年末予定 |
| 規模 | ギガワット級データセンター |
| ネットワーク | Broadcom Tomahawk イーサネット |

### 設計思想

OpenAIハードウェア部門のRichard Ho氏は、「JalapeñoはフロンティアAIモデルにとって最も重要なカーネル、メモリ移動、ネットワーク、サービングパターンを中心に最適化された」と説明している。

このチップの核心的な設計思想は3つ：

1. **推論特化**: 汎用アクセラレーターではなく、LLM推論のメモリボトルネックとレイテンシに特化
2. **フルスタック統合**: OpenAIのモデル知見をハードウェアに直接組み込む
3. **多世代路線図**: 最初の一歩であり、2029年までに10GW規模への拡張を計画

## なぜこれが重要か

### 1. AI推論コストの根本的変革

業界試算では、1GWデータセンターのコストは約500億ドル、うち約350億ドルがチップに割かれる（Nvidia現在の価格基準）。OpenAIが自社チップを持つことで、推論コストを大幅に削減できる可能性がある。

Sam Altman氏は「膨大な計算インフラを世界に提供できる。大きな効率向上につながり、より良いパフォーマンス、より速いモデル、より安いモデル——すべてが実現する」と述べている。

### 2. ハードウェア主権の獲得

「自分たちのチップを作れば、運命をコントロールできる」——Altman氏のこの言葉は、Nvidia依存からの脱却を象徴している。OpenAIは現在約2GWの計算容量で稼働しており、Nvidia、AMD、Broadcomとの合計で約33GWの計算コミットメントを発表済みだ。

### 3. 自社モデルでチップ設計を加速

Brockman氏は、「OpenAIのモデルを使ってチップ設計を加速し、効率を改善した。大規模な面積削減を実現した」と語っている。人間が最適化済みのコンポーネントに計算力を注ぎ込むことで、モデル独自の最適化が生まれるという。

## Broadcomとの協力関係

この発表は2025年10月に発表された10GW共同開発計画の具体的成果だ。

- **Broadcom**: シリコン実装、ネットワーク（Tomahawk）、接続技術を提供
- **Celestica**: ボード、ラック、システム統合を担当
- **OpenAI**: チップ設計とアーキテクチャを主導

Hock Tan氏（Broadcom CEO）は「これは多世代にわたる路線図の始まりに過ぎない。2026年末からMicrosoftや他のパートナーとのギガワット級データセンターにデプロイを開始する」と述べている。

## 業界への影響

### Nvidiaへの影響

Jalapeñoの発表は、NvidiaのAIアクセラレーター市場への直接的な挑戦だ。OpenAIはNvidiaと10GWのコミットメント（最大1000億ドル投資）を持っているが、自社チップの成功は将来的にNvidiaへの依存を減らす可能性がある。

### カスタムチップ競争の激化

Google（TPU）、Amazon（Trainium/Inferentia）、Meta（MTIA）に続き、OpenAIもカスタムAIチップの競争に参入。AI業界では「自社チップを持つこと」が差別化要因になりつつある。

### 推論コストの構造変化

Jalapeñoが「現在のSOTAより大幅に良い性能ワット比」を実現すれば、LLM APIの価格構造に根本的な変化をもたらす可能性がある。特にChatGPTのような大規模サービスでは、推論コスト削減が直接的な利益向上につながる。

## まとめ

Jalapeñoは単なるAIチップの発表ではない。OpenAIが「モデル会社」から「フルスタックAIプラットフォーム」へ移行するための重要な一歩だ。9ヶ月という驚異的な開発スピード、自社モデルを活用したチップ設計、10GW規模のデプロイ計画——これらはAI業界のインフラ競争が新たな段階に入ったことを示している。

2026年末の初回デプロイ以降、JalapeñoがLLM推論のコストとパフォーマンスにどのような影響を与えるか、注目していく必要がある。
